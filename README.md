## 🛡️ Sobre o Bunker

Sou um desenvolvedor focado em construir soluções **robustas**, **privadas** e **soberanas**. Atualmente, curso Análise e Desenvolvimento de Sistemas, unindo a criação de software (Backend) com a segurança e o suporte de infraestrutura. 

Acredito que o código aberto não é apenas uma escolha técnica, mas uma necessidade ética para garantir a liberdade digital.

- 🌍 **Localização:** Pernambuco, Brasil.
- 🔐 **Foco:** Desenvolvimento Seguro, Automação de Pipelines e Hardening de Sistemas.
- 🐧 **Filosofia:** "O software é como a liberdade: você só a tem se puder ver como ela funciona."
