## Sobre mim

Atuo no desenvolvimento backend, com foco em aplicações web e integração com banco de dados. Estou em formação em Análise e Desenvolvimento de Sistemas, conciliando prática de desenvolvimento com fundamentos de infraestrutura e suporte.

Tenho interesse em segurança, redes e na construção de sistemas que funcionem de forma simples, confiável e bem estruturada.

Fã assíduo pelos projetos de Software Livre e código aberto.
